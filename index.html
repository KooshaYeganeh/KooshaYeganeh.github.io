Recomended Linuxes


Fedora


Linux Fedora is the most popular Linux that I tested. I can mention several reasons for this: 1- The first point, when you are both a developer and a system administrator, in this case, many serious servers in the world are rpm, and when you work with Fedora every day, it is as if you are sitting behind a cemtos or rocky server. 2- Fedora is very smooth and stable 3- Interesting projects are implemented on Fedora. For example, the Fedora spins project. 4- You can find a lot of information about Fedora on the Internet. As someone who is both a developer and a system administrator, the best Linux I have tested is Fedora.


openSUSE


If Fedora is the Lamborghini of Linuxes, then the slogan that most of those who use SUSE is Mercedes-Benz Linux is completely correct.suse linux is a very stable and fast linux. When I tested it, it was installed by default in the installation process with the btrfs file system. This means it is faster than the others. Many people who are developers, when they compare Ubuntu with Suse, they vote for Ubuntu, but my opinion is completely different. I tested open suse with kde and xfce desktops and they work very stable and smooth. In my opinion, one of the strengths of SUSE Linux is the company and the community that is behind this operating system. SUSE has one of the largest contributions to the Linux kernel and large projects such as Open Stack, and this is clearly visible in the quality of work with SUSE. to be My only problem with suse at first is the manager package and adding the PDP repositories when installing the software. (I also solved this problem by writing a package manager like in SUSE 15.4) suse is one of the most popular


BunsenLabs


The only thing I want to say about bunsenlabs is: minimal, fast and easy to use The only problem I had is that, like Ubuntu, you cannot count on supporting all packages and solving problems


CuteFish


It was one of the most beautiful Linuxes I tested. If you like Ubuntu and want a different Ubuntu, I recommend cutefish. Only one big problem that happened to me was its strange slowness, which was really painful after a while


Rocky


A worthy successor to centos


Lubuntu


In my opinion, if you want to use Ubuntu, you should use Lubuntu instead. Even I prefer lubuntu because of its simplicity and ease of use.


Debian


The best alternative for versions like centos and... I tested Debian both on a server and on a personal laptop. I used Gnome desktop on my personal laptop. In my opinion, it is the best non-rpm Linux


Manjaro


It is not a suitable option for someone who is a developer and system administrator, but the experience was completely different


Kali


In short, the best Linux for penetration testing


Kodachi


It was one of the strangest and most interesting Linuxes that I tested. First of all, I can say that it can be used as an operating system for normal tasks, and there are no special problems in this regard. It is really safe and for someone who cares about his safety and his work, I think it is one of the best options. There are similar options, but none of them are as good as Kodachi.


Puppy

I tested puppy compatible with ubuntu. There is a different experience in using Linux. In my opinion, everyone who likes Linux should try puppy
